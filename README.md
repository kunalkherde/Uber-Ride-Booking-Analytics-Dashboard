# 🚖 UBER Ride Booking Analytics Dashboard (Power BI)

## 📌 Project Overview
This project presents a comprehensive analysis of a ride booking dataset using Power BI. The dashboard provides insights into booking trends, revenue performance, customer behavior, and operational efficiency.

The goal of this project is to transform raw ride data into meaningful business insights that help in decision-making and performance optimization.

---

## 🎯 Objectives
- Analyze booking trends and demand patterns  
- Identify peak hours and high-demand locations  
- Understand cancellation behavior and reasons  
- Evaluate revenue performance and efficiency  
- Analyze customer and driver ratings  

---

- Total Bookings, Revenue, Completion Rate  
- Booking & Revenue Trends  
- Ride Status Distribution

### 5. Customer & Driver Insights

---

## 📈 Key Insights
- Peak bookings occur during evening hours  
- Auto and bike rides contribute significantly to total bookings  
- Majority of rides are successfully completed  
- Certain locations show higher cancellation rates  
- Customer ratings are generally high but vary by vehicle type  

---

## 📂 Dataset
The dataset contains ride booking details including:
- Booking ID, Date & Time  
- Pickup & Drop Locations  
- Ride Distance & Booking Value  
- Booking Status  
- Customer & Driver Ratings  
- Payment Methods
